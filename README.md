# 🌶️ Spice Garden Restaurant Website

A modern and responsive website for **Spice Garden Restaurant**.  
This project showcases the restaurant’s menu, gallery, reservation system, and contact details in a clean and user-friendly design.

---

## 📌 Features

- 🏠 **Homepage** with hero banner and restaurant introduction
- 🍽️ **Menu Section** with categorized food items
- 👨‍🍳 **About Us** section describing the restaurant story
- 📸 **Gallery** showcasing dishes and restaurant interiors
- 📅 **Table Reservation Form**
- 📞 **Contact Section** with location and contact details
- 📱 **Responsive Design** for mobile, tablet, and desktop
- 🎨 Modern UI with spice-inspired colors

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling and layout  
- **JavaScript** – Interactive elements  
- **Responsive Design** – Mobile-friendly layout
